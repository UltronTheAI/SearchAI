## Installation In Your WorkSpace

Install my-project with npm

```bash
git clone https://github.com/UltronTheAI/SearchAI.git
cd SearchAI
npm i
npm start
```
Create a .env file in project dir

```env
PORT=5000
ID=*************
```

In this file, the `ID` is `Google Bard __Secure-1PSID`
